# eos
The Eos project creates a graph from a stream of supplied user input data

## Executive Summary
The ultimate objective of Eos is to seamlessly integrate streams of user input data into a graphical format ready to be stored in a graph database.  
Auxiliary goals include analytics and inference of graphic data.

## Current State
The first stage of Eos is to create a graph from a stream of data without either importing nor exporting a graph.
